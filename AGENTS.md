# Datos de utilidad para agentes

Graphily ya esta instalado en el dispositivo solo usa el comando ```graphily```.

Este proyecto tiene graphily como utilidad para el analisis de codigo
- .agents: Contiene las instrucciones para los agentes
- graphify-out: Contiene los resultados de graphily
- graphify-out/UESC-API-callflow.html: Contiene el callflow del proyecto

Todo los agentes deben trabajar con el metodo de graphily para realizar sus tareas.
